# SteakFrames
### World of Warcraft: Wrath of the Lich King

![Screenshot showing raid frames](https://trap-nine.com/screenshots/WoWScrnShot_030926_050408.jpg)

Show frames for party, raid (10-man), player, target, target-of-target, focus and pets.
Uses GearScore addon to show gearscores on unit frames.

These frames are still a work in progress.
